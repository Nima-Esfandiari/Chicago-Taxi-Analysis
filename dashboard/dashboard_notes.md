# Dashboard Overview

This dashboard was built using Tableau Public.

## KPIs

- Total Trips
- Total Revenue
- Average Trip Distance
- Average Revenue per Trip

## Charts

- Annual Revenue Trend
- Annual Trip Volume
- Revenue by Payment Type
- Average Trip Distance Trend

## Purpose

The dashboard provides an overview of Chicago taxi trip performance from 2013–2023 and allows users to quickly identify revenue trends, trip volume changes, payment preferences, and travel distance patterns.
