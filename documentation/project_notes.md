# Project Notes

## Objective

Analyze the Chicago Taxi Trips public dataset using SQL in Google BigQuery and visualize the results with Tableau Public.

## Project Workflow

1. Connected to the Google BigQuery Public Dataset.
2. Explored the dataset and identified key business questions.
3. Wrote SQL queries to answer each business question.
4. Validated query results in BigQuery.
5. Built visualizations in Tableau Public.
6. Combined the visualizations into an interactive dashboard.
7. Published the dashboard.
8. Documented the project on GitHub.

## Tools Used

- Google BigQuery
- SQL
- Tableau Public
- GitHub

## Repository Structure

- `sql/` – SQL queries
- `images/` – BigQuery screenshots
