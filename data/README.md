# Data

## Dataset

This project uses the Google BigQuery Public Dataset:

`bigquery-public-data.chicago_taxi_trips.taxi_trips`

### Dataset Information

- Source: Google BigQuery Public Dataset
- Records analyzed: Over **213 million** taxi trips
- Location: Google Cloud BigQuery

The raw dataset is not included in this repository because of its large size.

You can access the dataset here:

[https://console.cloud.google.com/marketplace/product/city-of-chicago-public-data/chicago-taxi-trips]
